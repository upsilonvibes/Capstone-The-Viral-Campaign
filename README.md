# Moringa Glow: The Viral Campaign 🌿✨

A high-end, responsive landing page showcasing a 7-day viral marketing strategy for a premium Kenyan skincare brand. This project merges the "Quiet Luxury" aesthetic with ancestral botanical wisdom, specifically focused on cold-pressed Moringa oil from the Rift Valley.


## 📖 Project Vision
**Moringa Glow** is designed as a "Cultural Bridge"—taking the Shamba-to-Skin philosophy of Kenya and presenting it for a global, modern audience. The project serves as a showcase for digital storytelling, web development, and strategic content planning.

### Key Pillars:
*   **Ethically Sourced:** Direct partnerships with small-scale farmers in Narok.
*   **High Performance:** Naturally rich in Vitamins A, C, and E.
*   **Quiet Luxury:** Understated elegance and sustainable glass packaging.

---

## 🛠️ Tech Stack
*   **Frontend:** Semantic HTML5, CSS3 (Custom Variables & Flexbox/Grid)
*   **Typography:** Playfair Display (Serif) & Inter (Sans-serif)
*   **Hosting:** Firebase Hosting
*   **Version Control:** Git & GitHub (Managed via System CMD)

---

## 📅 The 7-Day Viral Strategy
The landing page includes a dedicated section for a week-long social media campaign:
1.  **Day 01:** The ASMR Unboxing (Visual Hook)
2.  **Day 02:** Green Gold Thread (Education)
3.  **Day 03:** Clean Girl Routine (Lifestyle Integration)
4.  **Day 04:** Ancestral Secrets (Heritage/Storytelling)
5.  **Day 05:** Traceability Journey (Trust & Transparency)
6.  **Day 06:** Mindful Minimalist (Brand Philosophy)
7.  **Day 07:** Global Reach (Community & Growth)
